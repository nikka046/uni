# Vim Cheatsheet

This is my vim cheatsheet. enjoy.

 * exit insert mode: `ESC`
 * enter insert mode: `a`

 * end of line: `$`
 * start of the line: `^`

 * start selecting: `v`
 * select whole line: `V`
 
 * delete selection: `d`
 * delete whole line: `dd`
 
 * move one line down: `j`
 * move one line up: `k`
 * move one to the left: `h`
 * move one to the right: `l`

 * copy selection: `y`
 * paste selection: `p`

 * undo: `u`
 
 * move current line up: ``
 * move current line down: ``
